# Simple Jump Page

A simple HTML page that allows you to quickly jump to websites by entering URLs.

## Features

- Enter any website URL and jump to it
- Quick links to common local development ports (3000, 3333, 5173)
- Mobile responsive design
- Clean and simple interface

## Deployment

This project is designed for easy deployment on Vercel:

1. Push to GitHub
2. Connect to Vercel
3. Deploy automatically

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using Bun
bun serve .
```
